## 为什么需要GEO追踪工具？

传统SEO有Google Analytics、Ahrefs、SEMrush等工具来追踪排名和流量。但在AI搜索时代，这些工具无法告诉你：

- 你的品牌在AI回答中出现的频率
- AI如何描述你的产品或服务
- 竞争对手在AI搜索中的表现如何
- 你的内容是否被AI引用为信息源

<Badge color="orange">关键数据</Badge> 根据研究，超过**60%**的用户开始使用AI搜索引擎来获取产品信息和购物建议。如果你的品牌不在AI的回答中，你就失去了这部分潜在客户。

## 可见性评分算法：科学量化AI表现

这个工具最核心的价值是它的**可见性评分算法**（0-100分）。这个分数不是简单的"出现/不出现"，而是综合考虑了5个维度：

| 维度 | 说明 |
|------|------|
| **Mention（提及）** | 品牌名称是否在回答中出现 |
| **Position（位置）** | 出现在回答的哪个位置（开头/中间/结尾） |
| **Frequency（频率）** | 在多个相关提示词中出现的次数 |
| **Citation（引用）** | 你的网站是否被引用为信息源 |
| **Sentiment（情感）** | AI对你的品牌描述是正面/中性/负面 |

<Badge color="blue">评分示例</Badge> 如果你的品牌在10个相关提示词中被提及8次，每次都在回答的前半部分，且网站被引用了5次，描述都是正面的，你的可见性分数可能达到**85分**。

## 技术栈与部署

### 技术特点

- **本地优先**：所有数据存储在本地（IndexedDB + localStorage），隐私安全
- **现代框架**：Next.js 16.1 + Turbopack，性能优秀
- **多AI支持**：ChatGPT、Perplexity、Gemini、Copilot、Google AI Overview、Grok
- **自动化**：支持Cron定时任务和GitHub Actions

### 快速部署步骤

<Steps>
  <Step>
    **克隆项目**

    ```bash
    git clone https://github.com/danishashko/geo-aeo-tracker
    cd geo-aeo-tracker
    ```
  </Step>

  <Step>
    **安装依赖**

    ```bash
    npm install
    ```
  </Step>

  <Step>
    **配置API密钥**

    复制环境变量模板：

    ```bash
    cp .env.example .env.local
    ```

    然后编辑`.env.local`，添加以下API密钥：
    - **Bright Data API密钥**：用于抓取AI搜索引擎
    - **OpenRouter API密钥**：用于调用AI模型

    <Callout type="warning">
    Bright Data是付费服务，但提供免费试用。OpenRouter支持多种AI模型，按使用量付费。总体成本远低于商业GEO工具。
    </Callout>
  </Step>

  <Step>
    **启动服务**

    ```bash
    npm run dev
    ```

    访问 `http://localhost:3000` 即可使用。
  </Step>

  <Step>
    **（可选）部署到生产环境**

    ```bash
    vercel --prod
    ```
  </Step>
</Steps>
