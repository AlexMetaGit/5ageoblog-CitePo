# llms.txt 标准解读：AI 搜索时代的 robots.txt

<Callout type="info">
  <strong>早间导读</strong>：ChatGPT 占据 AI 推荐流量的 87.4%，但 24% 的响应无需获取在线内容。如何让 AI 找到并信任你的品牌？答案可能是——在你的网站根目录放一个 llms.txt 文件。
</Callout>

## 为什么传统 SEO 正在失效？

过去二十年，SEO 的核心逻辑是：**关键词 + 反向链接 = 排名**。

但在 AI 搜索时代，这个公式正在瓦解：

<Badge color="blue">数据说话</Badge>

- 用户向 ChatGPT 输入 **23 个单词** 的提示词，而传统搜索平均只有 **4.2 个单词**
- **ChatGPT 占 AI 推荐流量的 87.4%**，但转化率是传统来源的 2 倍
- 内容更新时间 **< 3 个月** 的被引用概率高 **3 倍**
- **24%** 的 ChatGPT 响应无需获取任何在线内容（直接基于训练数据回答）

这意味着什么？**权威现在存在于 AI 模型内部**，而不仅仅是网页上。

## 什么是 llms.txt？

llms.txt 是由 Answer.ai 提出的标准规范，类比 robots.txt，但专门为大型语言模型设计。

**核心作用**：帮助 ChatGPT、Claude、Gemini 等 AI 引擎更好地理解和引用你的网站内容。

<Badge color="green">标准规范</Badge>

| 特性 | 说明 |
|------|------|
| **文件位置** | 网站根目录 `/llms.txt` |
| **扩展版本** | `/llms-full.txt`（完整内容索引）|
| **格式** | Markdown + YAML 元数据 |
| **许可证** | MIT（开源免费）|
| **GitHub Stars** | 2,216 ⭐（GEO/AEO 领域排名第 1）|

## llms.txt 文件结构详解

一个标准的 llms.txt 文件包含三个部分：

```markdown
---
title: 你的网站名称
description: 简短描述，告诉 AI 你的网站是做什么的
version: 1.0
updated: 2026-04-20
---

# Content Sections

## 产品文档
- 快速开始: /docs/getting-started
- API 文档: /api/reference
- 使用教程: /docs/tutorials

## 核心数据
- 数据集: /data/dataset.json
- 模型权重: /models/latest
- 案例研究: /cases
```

### 关键设计原则

<AccordionGroup>
  <Accordion title="1. 语义清晰度优先">
    AI 模型偏好结构清晰、意义明确的内容。使用描述性标题，避免模糊表达。
  </Accordion>
  <Accordion title="2. 实体建模">
    明确标注人名、品牌、技术术语等实体，帮助 AI 建立知识关联。
  </Accordion>
  <Accordion title="3. 对话式结构">
    使用问答格式组织内容，匹配用户向 AI 提问的方式。
  </Accordion>
</AccordionGroup>

## 实施步骤：5 分钟快速部署

<Steps>

### 第一步：创建文件

在网站根目录创建 `llms.txt`：

```bash
touch /var/www/html/llms.txt
```

### 第二步：添加元数据

使用 YAML frontmatter 格式：

```yaml
---
title: 你的品牌名称
description: 一句话描述你的核心业务
version: 1.0
updated: 2026-04-20
---
```

### 第三步：组织内容索引

按类别整理你的核心内容：

```markdown
## 文档
- 产品文档: /docs
- API 参考: /api
- 博客: /blog

## 数据资源
- 数据集: /data
- 研究报告: /research
```

### 第四步：验证可访问性

确保文件可通过 HTTPS 访问：

```bash
curl -s https://your-domain.com/llms.txt
```

### 第五步：定期更新

设置提醒，每月检查并更新内容索引。

</Steps>

## GEO 的核心理念：信任而非排名

传统 SEO 关注**排名和点击**，GEO 关注**引用和信任**。

**关键认知转变**：

<Badge color="red">核心洞察</Badge>

> "不是关键词或反向链接，是确保 AI 信任并引用你的内容。引用是新点击：被引用意味着被发现。"

### E-E-A-T 在 AI 时代的新含义

| 传统 SEO | GEO/AEO |
|---------|---------|
| Experience（经验）| 训练数据中的出现频率 |
| Expertise（专业性）| 被权威来源引用的次数 |
| Authoritativeness（权威性）| 跨平台一致的品牌信号 |
| Trustworthiness（可信度）| 事实准确性和引用质量 |
