## 为什么需要 GEO/AEO 工具？

传统 SEO 时代，我们有 Google Analytics、Search Console 等成熟工具。但在 AI 搜索时代，这些工具无法回答关键问题：

- **品牌在 ChatGPT 回答中被提及了吗？**
- **用户问相关问题时，AI 会推荐你的产品吗？**
- **与竞争对手相比，你的 AI 可见性如何？**

这就是 GEO/AEO 工具存在的意义。

<Badge color="blue">核心价值</Badge> GEO/AEO 工具帮助你**量化 AI 时代的品牌可见性**，从"感觉被提及"到"数据驱动优化"。

## 实战：从零部署 geo-aeo-tracker

<Steps>

### 克隆项目

```bash
git clone https://github.com/danishashko/geo-aeo-tracker
cd geo-aeo-tracker
```

### 安装依赖

```bash
npm install
```

### 配置环境变量

```bash
cp .env.example .env.local
```

编辑 `.env.local`，添加以下 API 密钥：

- `BRIGHT_DATA_API_KEY`：用于抓取 AI 平台响应
- `OPENROUTER_API_KEY`：用于 LLM 分析

### 启动开发服务器

```bash
npm run dev
```

访问 `http://localhost:3000` 开始使用。

### 创建首个项目

1. 进入"项目设置"标签页
2. 填写品牌名称、关键词、描述
3. 添加追踪提示词（建议 10-30 个/主题集）
4. 运行首次抓取

</Steps>

## 可见性评分算法解析

geo-aeo-tracker 的可见性分数（0-100）基于以下维度：

| 维度 | 权重 | 说明 |
|------|------|------|
| **Mention（提及）** | 25% | 品牌是否被提及 |
| **Position（位置）** | 20% | 在回答中的位置（前/中/后） |
| **Frequency（频率）** | 20% | 提及次数 |
| **Citation（引用）** | 20% | 是否引用你的网站 |
| **Sentiment（情感）** | 15% | 积极/中立/消极 |

<Callout type="tip">
  <strong>HubSpot 实战经验</strong>：来自 LLM 的流量转化率是传统搜索的 **3 倍**。原因是用户在到达网站前已通过 AI 完成大量研究，购买意图更强。
</Callout>

## HubSpot 的 AEO 工具使用策略

根据 HubSpot 全球增长总监 Aja Frost 的分享，HubSpot 在经历 **80% 流量下降** 后，通过系统化的 AEO 策略成为 **LLM 中最被引用的 CRM**。

### 四大北极星指标

HubSpot 的工具追踪重点：

1. **可见性（Visibility）** - 最重要
   - 品牌在 AI 回答中的出现频率
   - 最终驱动转化

2. **声量份额（Share of Voice）**
   - 相对于竞争对手的可见度
   - 抵消算法变化影响

3. **引用次数（Citations）**
   - 网站作为答案来源被引用
   - 有引用 → 更好的排名和情感

4. **提及次数（Mentions）**
   - 品牌被提及的频率

### 追踪频率建议

<Badge color="purple">最佳实践</Badge> 月度追踪最佳，季度最低频率。LLM 更新训练数据、权重和响应生成模式的频率比传统搜索算法更高。

## 常见问题

<AccordionGroup>
  <Accordion title="Q: 需要多少预算？">
    所有工具都是开源免费的，只需要 API 密钥费用。Bright Data 约 $50-100/月，OpenRouter 按使用量计费（通常 $10-30/月）。
  </Accordion>
  <Accordion title="Q: 多久能看到效果？">
    通常 1-3 个月能看到可见性提升，3-6 个月能有显著效果。这取决于你的内容质量和优化力度。
  </Accordion>
  <Accordion title="Q: 可以不用付费 API 吗？">
    可以尝试手动追踪：标准化提示词集 → 跨主要 AI 引擎测试 → 记录提及和引用 → 月度审查结果。但效率较低，建议小规模测试时使用。
  </Accordion>
  <Accordion title="Q: 如何处理 AI 结果的可变性？">
    将可变性视为预期特性。聚合每个提示词 5-10 个样本，记录平均提及率、情感和引用频率。
  </Accordion>
</AccordionGroup>
