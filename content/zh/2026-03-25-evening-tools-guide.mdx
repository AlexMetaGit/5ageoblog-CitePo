AI 搜索时代，传统 SEO 工具（如 Ahrefs、SEMrush）已经无法满足需求。你需要专门的工具来：

根据 HubSpot 的实战经验，AI 搜索可见性（Visibility）已成为最重要的北极星指标，直接驱动转化。
</Callout>

## 深度评测：geo-aeo-tracker

<Badge color="purple" icon="star">编辑推荐</Badge>

**项目地址**: https://github.com/danishashko/geo-aeo-tracker

### 核心优势

1. **功能全面**：12 个功能标签页覆盖所有 GEO 工作流
   - 项目设置、提示词中心、角色扩散
   - 细分领域探索器、响应浏览
   - 可见性分析、引用频率、竞争对手分析
   - AEO 审计、自动化任务

2. **本地优先**：数据不离开机器，隐私安全
3. **完全免费**：$0/月，开源
4. **多模型支持**：ChatGPT、Perplexity、Gemini、Copilot、Google AI Overview、Grok

### 可见性评分算法

分数（0-100）基于：
- **品牌提及**（Mention）
- **位置**（Position）
- **频率**（Frequency）
- **引用**（Citation）
- **情感**（Sentiment）

<Callout type="tip">
geo-aeo-tracker 适合需要系统化 GEO 工作流的营销团队和开发者，功能全面且完全免费。
</Callout>

### 配置要求

需要两个 API 密钥：
- **Bright Data API**：用于抓取 AI 平台响应
- **OpenRouter API**：用于 LLM 分析

### 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/danishashko/geo-aeo-tracker
cd geo-aeo-tracker

# 2. 安装依赖
npm install

# 3. 配置环境
cp .env.example .env.local
# 编辑 .env.local，添加 API 密钥

# 4. 启动服务
npm run dev
```

## 实战案例：HubSpot 的 AEO 转型

<Badge color="orange">案例研究</Badge>

HubSpot 经历了 **80% 流量下降**的危机后，全面转向 AEO（Answer Engine Optimization），建立了新的成功指标体系：

### 四大北极星指标

1. **VISIBILITY（可见度）** - 最重要
   - 品牌在 AI 回答中的出现频率
   - 直接驱动转化

2. **SHARE OF VOICE（声量份额）**
   - 相对于竞争对手的可见度
   - 抵消算法变化影响

3. **MENTIONS（提及次数）**
   - 品牌被提及的频率

4. **CITATIONS（引用次数）**
   - 网站作为答案来源被引用的次数

<Callout type="warning">
HubSpot 强调：引用次数排在第三位，不应过度追求。可见度和声量份额才是核心指标。
</Callout>

### HubSpot 的工具策略

HubSpot 使用内部工具追踪这些指标，但对于大多数品牌，geo-aeo-tracker 提供了类似的功能，且完全免费。
