# GEO工具评测：geo-aeo-tracker 深度体验

在 AI 搜索时代，**88% 的品牌在 AI 引擎中是不可见的**[来源: GEO-AEO-NotebookLM]。即使你在传统搜索中排名靠前，也不代表你能出现在 ChatGPT、Perplexity 或 Google AI Overviews 的回答中。

如何追踪你的品牌在 AI 搜索中的表现？今天我们来深度评测一个开源工具：**geo-aeo-tracker**。

<Callout type="info">
**工具定位**：本地优先、完全免费的开源 GEO 追踪仪表板，支持 ChatGPT、Perplexity、Gemini、Copilot、Google AI Overview 和 Grok 六大平台。
</Callout>

## 为什么需要 GEO 追踪工具？

传统 SEO 工具（如 Ahrefs、SEMrush）只能追踪传统搜索排名，但 AI 搜索的可见性完全不同：

- **传统搜索**：追踪关键词排名、点击率、流量
- **AI 搜索**：追踪品牌提及、引用频率、位置、情感

<Badge color="red">核心问题</Badge> 没有专门的工具，你无法知道品牌在 AI 回答中的表现如何。

## geo-aeo-tracker 核心功能

这个工具提供了 **12 个功能标签页**，覆盖完整的 GEO 工作流：

### 1. 项目设置 ⚙️
设置品牌名称、关键词、描述，创建追踪项目。

### 2. 提示词中心 💬
管理用于追踪的提示词库，支持批量测试。

### 3. 角色扩散 🎭
生成角色特定提示词，测试不同用户场景下的品牌可见性。

### 4. 细分领域探索器 🔍
生成高意图查询，发现长尾关键词机会。

### 5. 响应浏览 📝
浏览 AI 响应内容，查看品牌是否被提及。

### 6. 可见性分析 📊
**核心功能**：趋势图表展示可见性分数变化。

<Callout type="tip">
**可见性评分算法**（0-100分）基于：
- 品牌提及（Mention）
- 位置（Position）
- 频率（Frequency）
- 引用（Citation）
- 情感（Sentiment）
</Callout>

### 7. 引用分析 🔗
分析品牌被引用的频率和上下文。

### 8. 引用机会 🎯
**战略功能**：发现竞争对手的引用机会，找到你的内容缺口。

### 9. 竞争对手对战卡 ⚔️
AI 生成的品牌对比分析，了解你的优劣势。

### 10. AEO 审计 🏥
站点准备度检查，评估你的网站是否 GEO 友好。

### 11. 自动化 ⏱️
支持 Cron 和 GitHub Actions，实现定期自动追踪。

### 12. 文档 📖
14 部分完整指南，从安装到高级用法。

## 快速开始

### 步骤 1：安装

```bash
git clone https://github.com/danishashko/geo-aeo-tracker
cd geo-aeo-tracker
npm install
```

### 步骤 2：配置

```bash
cp .env.example .env.local
# 编辑 .env.local，添加 API 密钥
```

需要两个 API 密钥：
- **Bright Data**：用于抓取 AI 引擎响应
- **OpenRouter**：用于 LLM 分析

### 步骤 3：启动

```bash
npm run dev
# 访问 http://localhost:3000
```

### 步骤 4：部署

```bash
vercel --prod
```

## 最佳实践建议

### 初次使用
1. 创建项目，设置品牌信息
2. 添加 10-20 个追踪提示词
3. 运行首次批量抓取
4. 分析可见性分数基线

### 定期监控
1. 设置 Cron 定时任务（每周一次）
2. 监控分数变化趋势
3. 发现引用机会并优化内容
4. 对比竞争对手表现

### 数据驱动优化
- **可见性分数下降** → 检查内容质量和权威性
- **引用机会增加** → 创建针对性的内容
- **竞争对手领先** → 分析其优势并学习
